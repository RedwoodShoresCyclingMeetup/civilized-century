Gallery Options

The gallery page is currently under development and different implementations are available:

Gallery 1 - displays a small group of thumbnails as a grid. Clicking any thumbnail brings up a larger version of it. Left and right buttons allow the user to move between larger images. Navigating left on the first image takes you to the last and vice versa. The larger images can be navigated using the keyboard's left and right arrows.

Gallery 2 - displays one row of thumbnails which can be navigated through using left and right arrows. Navigating left on the first thumbnail takes you to the last and vice versa. The gallery periodically moves right one thumbnail on a loop until the user presses the pause, left or right buttons. If the autoplay is paused, the pause button turns into a play button to restart the automation. The larger images work as for Gallery 1.
Possible future improvements: adjust the size and positioning of elements to work properly on screens with small widths (such as cellphones); and implement touch gestures to allow thumbnails to be navigated by swiping.
